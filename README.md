## Git Aggregator

The very slow git commits aggregator written by bash


## Requirements

- bash v3.2.57 or later

## Getting started

```sh
$ cp config.bash.sample config.bash
$ vim config.bash
$ ./git-aggregator

$ cat commits.csv
Month,Count of commits
2018-10,4

$ cat diff.csv
Month,Additional lines,Deleted lines
2018-10,208,31
```
